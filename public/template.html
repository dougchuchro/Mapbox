Store Locator

Node.js GeoJSON API & App

Inspried by Traversy Media's Store Locator video on YouTube.
Node/Express/Mongo API with GeoJSON location field for store locations.
A simple vanilla JS frontend using the Mapbox Library.
I made some modifications including changing the GeoCoding fuction to use to Mapbox API rather than the Map Quest API.
